Tessie API Dashboard

Manage your Tesla vehicles via the Tessie API

API Endpoints

GET /status?vin=YOUR_VIN

Get current vehicle state

POST /wake

Wake up the vehicle

POST /command/start_charging

Start charging the vehicle

POST /command/set_charge_limit

Set charge limit (requires percent parameter)

Authentication

All API requests require a JWT bearer token in the Authorization header:

Authorization: Bearer YOUR_JWT_TOKEN

Generate a token using the provided JWT utilities in the repository.

Quick Start

  1. Configure your JWT secret: wrangler secret put JWT_SECRET
  2. Configure your Tessie API key: wrangler secret put TESSIE_API_KEY
  3. Generate a JWT token using the helper module
  4. Make API requests with your token

Example Request

curl -X GET "https://your-worker.workers.dev/status?vin=YOUR_VIN" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
